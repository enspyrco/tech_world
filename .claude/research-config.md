# Research Configuration

## Project Context
Tech World - Educational multiplayer game where players solve coding challenges together with an AI tutor (Claude bot).

## Tech Stack
- **Flutter/Dart**: Flame game engine, LiveKit client
- **Firebase**: Auth, Cloud Functions
- **WebSocket**: Real-time multiplayer
- **LiveKit**: Proximity-based video chat

## Research Priorities

- A* pathfinding algorithms
- LiveKit Flutter integration
- AI tutoring and hint systems
- Multiplayer game synchronization
- WebSocket best practices

## Preferred Sources

- LiveKit documentation
- Firebase documentation
- Game development tutorials and patterns
