## Slide Style Preferences
- Clean, game-inspired aesthetic
- Use screenshots/diagrams of the game world
- Include architecture diagrams for technical audiences
- Dark primary background for title slides
- White background for content slides

## Common Presentation Types

### Technical Overview
1. Title + project description
2. Architecture diagram (client, server, Firebase)
3. Game world and Flame engine
4. Multiplayer networking flow
5. AI tutor integration plans

### Status Update
1. Title + date
2. Completed work
3. Current blockers
4. Upcoming tasks
5. Demo or metrics

### Feature Deep-Dive
1. Feature title
2. Problem / motivation
3. Technical approach
4. Implementation details
5. Results / demo
